### 1.[**webstorm** 在**Vue**项目下不识别**@**别名](https://segmentfault.com/a/1190000022495271)

在 **webstorm File** > **Settings** > **Languages & Frameworks** > **JavaScript** > **Webpack** 下选择当前项目下的

```
"node_modules/@vue/cli-service/webpack.config.js"
```
